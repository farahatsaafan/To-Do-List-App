# ✅ To-Do List Application

A Microsoft To-Do inspired task management desktop application
built with C# and Windows Forms using the Guna UI2 library.

> 👥 Team Project — New Mansoura University, CS & AI Department
> 💼 My Contribution: Database & Storage layer

## 🛠️ Technologies Used
- C# (.NET)
- Windows Forms
- Guna UI2 (Modern UI Components)
- Object-Oriented Programming

## ✨ Features
- **My Day** — Shows today's tasks, tasks older than 24hrs 
  move automatically to Suggestions
- **Important** — Star-marked priority tasks
- **Planned** — Scheduled tasks visible across My Day & All
- **All** — Combined view of all task categories
- **Tasks** — Private tasks not shown in other views
- **Complete** — Archive of finished tasks
- Task completion with strikethrough effect and color change
- Real-time date display

## ⚙️ My Contribution
- Designed and implemented the data storage structure for tasks
- Built persistent task properties:
  IsCompleted, IsImportant, IsPlanned, IsMyDayTask, IsAllTask
- Implemented task state management and filtering logic
- Ensured data integrity across all task categories

## 🚀 How to Run
1. Open `oopf.sln` in Visual Studio
2. Install Guna UI2 package via NuGet
3. Build and run the project
